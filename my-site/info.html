Цитаты в HTML

HTML предоставляет специальные теги для оформления цитат:

  • <blockquote> — используется для длинных цитат, оформляется как отдельный блок;
  • <cite> — указывает источник цитаты (книга, автор, сайт);
  • атрибут cite — содержит ссылку на источник, но визуально не отображается.

Пример оформления цитаты с источником:

«Электронная вычислительная машина – комплекс технических средств, предназначенных для автоматической обработки информации.»
Источник: Википедия

Медиаконтент в HTML

HTML позволяет встраивать мультимедиа без сторонних плагинов:

  • <video> — добавление видео;
  • <audio> — добавление аудио;
  • <iframe> — встраивание внешнего контента (YouTube, карты и т.д.);
  • <source> — указывает источники медиафайлов.
